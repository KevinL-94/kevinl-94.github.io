Markdown学习笔记

发表于 2017-03-11 | 分类于 IT | | 阅读次数

Markdwon，一种轻量级标记语言，程序员、文本编辑者必备技能。

什么是Markdown

首先，Markdown是一个格式化语法结构，目的是希望给大家提供一种易阅易写的纯文字格式实现快速排版。
其次，他是创始人Gruber编写的一个Perl脚本：Markdown.pl，用来实现Markdown至HTML转换的工具,即其本质为HTML标记语言。

那么什么地方要用到Markdown呢？

Markdown可以广泛地应用在博客、论坛等网络空间，实现快速便捷的排版。很多网站的文本编辑区都自带了Markdown编译器，输入Markdown代码后直接将其转化为HTML，就可以交给浏览器渲染了。

之所以说Markdown是程序员、文本编辑者的必备技能，就是因为Stack Overflow、Github、简书、微信平台这些主流代码、文本沟通社区都大量的使用了Markdown进行文本编辑排版。
